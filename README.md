# Chat Application

A simple chat application using react.js, express.js and socket.io

**Installation**

```bash
# frontend
$ cd frontend
$ npm install
# backend
$ npm install
```

## To view demo

```bash
# run frontend (:3000) and backend (:5000) concurrently
$ npm run dev
# run only frontend
$ npm run client  # http://localhost:3000
# run only backend
$ npm run serve  # http://localhost:5000

```
